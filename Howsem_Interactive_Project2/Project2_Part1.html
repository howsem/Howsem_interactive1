Play Controls
Play control is a approach that allow one to start or stop current action. It is usually used to control linear elements.
These symbols were originally used for the tape decks. Early tape decks didn't have these symbols. In between 1961 to 1964, these symbols became standardized.
The play symbol points to the direction in which the tape will move. In models where you load the cassette upside down, the arrow points to the left. Some tape decks have two play buttons (right pointing arrow and left pointing arrow), one for each side of the cassette. It adheres to the direction in which the tape is moving. The stop symbol is the play symbol without a pointing arrow which indicates that the tape is not moving.
Earlier versions of the tape deck didn't have a pause feature. Recording with stop and record left a noise because the write head was taken off and put back on the tape unlike when you pause. The pause button was mostly needed for tape recorders and not for cassette players. When the pause feature was introduced, a modified stop symbol was used.
